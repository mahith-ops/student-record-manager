# Student Record Manager

A full-stack **MERN** (MongoDB, Express, React, Node.js) web application built using the **MVC** architecture to manage student records efficiently.

## Features

- Create new student records (name, age, subjects, marks)
- View and display all students in a dashboard
- Update existing student information
- Delete student records
- Filter and search students by name, age, or subject

---

## Tech Stack

| Layer    | Tech                       |
| -------- | -------------------------- |
| Backend  | Node.js, Express.js (MVC)  |
| Database | MongoDB, Mongoose          |
| Frontend | React (CRA or Vite), Axios |
| Styling  | Material UI (MUI)          |
